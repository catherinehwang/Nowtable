Nowtable by khwang

Totally not Turntable.fm

An open-source clone of Turntable.fm.

Dependencies

Node, Express, Now, Connect-Form, Formidable

Install

Just make sure you have Node and all the requisite packages installed and then simply run it with sudo node server.js.
